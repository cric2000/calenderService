Calendry

API documentation for calendar service.

  • /add Endpoint

    Use the /add endpoint to calculate a new date by adding a specified number of days or weeks to the current date. If no date is provided, the operation will be performed on the current date.

    Parameters:

    • type: Specify whether to add "days" or "weeks".
    • value: The number of days or weeks to add.
    • date (optional): If provided, the calculation will be based on this date; otherwise, it defaults to the current date.

  • /sub Endpoint

    Use the /sub endpoint to calculate a new date by subtracting a specified number of days or weeks from a given date. If no date is provided, the operation will be performed on the current date.

    Parameters:

    • type: Specify whether to subtract "days" or "weeks".
    • value: The number of days or weeks to subtract.
    • date (optional): The base date from which subtraction occurs; defaults to the current date if not provided.

  • Examples

    Here are some example requests:

    • To add 1 week to the current date: /add?type=weeks&value=1
    • To subtract 5 days from a specific date: /sub?type=days&value=5&date=11-Feb-2023
